# 📄 JSON Formatter

A simple Python CLI tool that reads a JSON file and prints it in a clean, formatted way.

## ✨ Features

- Read JSON files
- Pretty-print JSON
- Detect invalid JSON
- Beginner-friendly

## ▶️ Run

```bash
python main.py
```

## 📌 Example

Input:

```json
{"name":"Aliya","age":20}
```

Output:

```json
{
    "name": "Aliya",
    "age": 20
}
```

## 🛠 Built With

- Python
- json (built-in library)
